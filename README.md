# ☕ 빽다방 키오스크 시스템 구현 (Python)

> **팀 프로젝트 (3조):** 파이썬을 활용한 키오스크 핵심 로직 구현 및 데이터 구조화
> **개발 기간:** 2026.04.20 ~ 2026.04.28

---

## 1. 프로젝트 개요
실제 빽다방 키오스크의 사용자 경험을 분석하여, 주문부터 결제까지의 전 과정을 Python으로 구현한 응용 소프트웨어 프로젝트입니다. 단순히 기능 구현에 그치지 않고, 팀원 간의 코드 통합을 위해 데이터 흐름을 설계하고 예외 상황을 최소화하는 데 집중했습니다.

## 2. 주요 구현 기능
* **메인 화면 및 진입 로직:** ASCII ART를 활용한 브랜드 로고 출력 및 포장/매장 선택 기능.
* **카테고리 시스템:** 메뉴의 효율적인 분류와 선택을 위한 동적 카테고리 로직.
* **주문 최적화:** 옵션 선택(온도, 크기, 샷 추가 등) 및 수량 조절 기능.
* **데이터 관리:** 장바구니 시스템 및 멤버십 적립/쿠폰 사용 로직.
* **최종 출력:** 실제 영수증 양식을 준수한 데이터 출력 및 랜덤 주문 번호 생성.

## 3. 담당 역할 및 기여 (정병인)
응용 소프트웨어 개발자로서 **프로그램의 기초 뼈대 설계와 사용자 인터페이스(CLI) 로직**을 주도했습니다.

* **메인 로직 설계:** 프로그램 시작 시 브랜딩을 위한 로고 구현 및 매장/포장 선택 분기점 설계.
* **카테고리 구조화:** 대규모 메뉴 데이터를 효과적으로 관리하기 위한 카테고리 분류 및 참조 로직 개발.
* **협업 및 코드 통합:** 팀원들이 각자 개발한 함수 간의 변수 명칭을 통일하고 전체적인 구조 조정 지원.
* **안정성 확보:** 사용자 입력값에 대한 `try-except` 예외 처리를 적용하여 안정적인 프로그램 구동 보장.

## 4. 핵심 코드 및 로직 설계
* **데이터 타입 활용:** 리스트와 딕셔너리를 중첩하여 메뉴명, 가격, 수량을 매칭하고 동적으로 관리.
* **순차적 흐름 제어:** `while` 루프와 조건문을 활용하여 '메뉴 선택 -> 장바구니 -> 결제' 로직 구현.
* **캐스팅(Casting):** 입력받은 문자열 데이터를 정수형으로 변환하여 금액 산출 로직에 적용.
